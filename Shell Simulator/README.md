## EEL5733 Shell Simulator

### Compile & Test

+ To build the project, run `make` (alias of `make all`).

+ Use executable test_* file to test cases

```
./test_c1 hello world > test_c1.txt
./test_c2 hello world < test_c2.txt
whoami | cat -
```
